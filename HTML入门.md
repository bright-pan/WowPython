HTML (HyperText Mark-up Language) <br>

# 2. html 基本结构

```
<!DOCTYPE html>
<html lang="zh-cn">

    <head>
        <meta charset="utf-8">
        <title>我的个人主页</title>
    </head>

    <body>
        hi guys  欢迎学习前端课程！
    </body>

</html>

```

| :--------- | :----- |

